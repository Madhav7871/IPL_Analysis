# 🏏 IPL Data Analysis

A data analysis project using Python to uncover key insights, team performances, and trends across Indian Premier League (IPL) seasons.

---

## 🚀 Key Objectives
* Analyze toss decisions and their impact on match outcomes.
* Identify top-performing teams and consistent players.
* Discover venue statistics (batting first vs. chasing advantages).

## 🛠 Tech Stack
* **Environment:** Google Colab
* **Language:** Python
* **Libraries:** `pandas`, `numpy`, `matplotlib`, `seaborn`

---

## 📈 Sample Insights
* Teams chasing targets have a higher win percentage at specific grounds.
* Toss winners choose to field first more than 60% of the time.
